## Overview

After v1.0 shipped the core room-visualization MVP, Jessica's direct feedback surfaced ~20 UX pain points: missing zoom, click-placement bugs, sticky tools, invisible save status, broken nav tabs, messy wall corners, and no way to edit placed elements. v1.1 fixes all of them — the app is now usable without workarounds and feels like a real CAD tool.

## Milestone Summary

**Key Decisions:**

- Cursor-anchored zoom math kept pure inside uiStore (zoomAt helper), not in canvas component
- `userZoom` + `panOffset` layer on top of auto-fit scale/origin — tools work unchanged at any zoom
- Wall rotation **detaches** from shared endpoints (simpler than dragging neighbors along)
- Product resize uses single `sizeScale` multiplier (aspect preserved, library dims never mutated)
- Corner caps: 2-wall junctions get precise outer-edge mitre; 3+ get convex hull; dead ends get square extension
- Opening handles appear when host wall is selected (no separate opening selection state)
- Generic `updateTextTag()` helper reused across product resize, wall endpoint/thickness drags, opening drags

**Issues Resolved:**

- Can't zoom on 2D canvas → scroll/pinch/buttons all work
- Click placement inaccurate → live preview shows where elements land
- Tools stay sticky → auto-revert to Select
- Length/size unknown until after → live tags during every drag
- Welcome screen had fake tabs → clean 2-CTA layout
- Save status invisible → prominent toolbar badge
- Walls at corners piled up → precise 2-wall caps + hull fills for 3+
- Dead-end walls had raw butts → square extension caps
- Crossing walls jagged → overlap-zone quads
- No way to edit placed walls → endpoint + thickness handles
- Can't resize doors/windows → side handles for width, center for slide

**Technical Debt Incurred:**

- Convex-hull caps for 3+ junctions aren't pixel-perfect at every angle (acceptable for floor plans)
- Opening handles only appear when host wall selected (simpler UX, no separate opening selection)
- No per-phase VERIFICATION.md files (each phase tested via Netlify deploy preview + PR test plan)
