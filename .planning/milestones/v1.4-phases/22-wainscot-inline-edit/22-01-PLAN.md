**Step A: Create `src/components/WainscotPopover.tsx`** (per POLISH-02)

A React component that renders the inline wainscot edit popover. Props:

```typescript
interface Props {
  wallId: string;
  side: WallSide;
  style: React.CSSProperties; // absolute position from FabricCanvas
  onClose: () => void;
}
```

Component body:
1. Read wall wainscoting config: `useCADStore(s => { const doc = s.rooms[s.activeRoomId!]; return doc?.walls[wallId]?.wainscoting?.[side]; })` -- subscribe to the specific wall's wainscot config so the popover re-renders on change.
2. Read style library: `useWainscotStyleStore(s => s.items)`.
3. Get `toggleWainscoting` from `useCADStore.getState()`.
4. Render a small card with the Obsidian CAD theme:
   - Header: `WAINSCOT EDIT` in `font-mono text-[10px] text-text-ghost`
   - Style dropdown: `<select>` following WallSurfacePanel pattern (lines 208-230). Value = `wainscotConfig.styleItemId ?? ""`. Options: `(LEGACY DEFAULT)` + all `items.map(it => it.name.toUpperCase() + " . " + STYLE_META[it.style].label)`. On change: call `toggleWainscoting(wallId, side, true, selected?.heightFt ?? config.heightFt, selected?.color ?? config.color, selectedId)`.
   - Height input: `<input type="number" step="0.25" min="1" max="8">` showing `wainscotConfig.heightFt`. On change: call `toggleWainscoting(wallId, side, true, newHeight, config.color, config.styleItemId)`.
5. Styling: `glass-panel` class for the container, `ghost-border`, width ~180px. Use same input/select classes as WallSurfacePanel: `font-mono text-[11px] bg-obsidian-high text-accent-light border border-outline-variant/30 px-1 py-0.5 rounded-sm`.
6. Dismissal:
   - `useEffect` with `mousedown` listener on `document`. If click target is outside the popover ref, call `onClose()`. Use a `ref` on the root div and check `!ref.current?.contains(e.target)`.
   - `onKeyDown` on the root div: if Escape, call `onClose()`. Make root div `tabIndex={-1}` and auto-focus it.
7. Dismiss on zoom/pan: `useEffect` that subscribes to `useUIStore` changes to `userZoom` and `panOffset`. If either changes while popover is open, call `onClose()`. Use `useUIStore.subscribe(...)` inside the effect and return the unsub.

2. Extend the EXISTING `mouse:dblclick` handler (lines 200-226). AFTER the dimension label hit-test loop (which returns early on hit), add wainscot wall hit-testing:
   ```typescript
   // After the existing for-loop that checks hitTestDimLabel...
   // Wainscot inline edit (POLISH-02)
   const feetX = (pointer.x - origin.x) / scale;
   const feetY = (pointer.y - origin.y) / scale;
   const feetPos = { x: feetX, y: feetY };
   for (const wall of Object.values(storeWalls)) {
     const { point } = closestPointOnWall(wall, feetPos);
     const d = distance(point, feetPos);
     if (d < 0.5 && wall.wainscoting) {
       const sideA = wall.wainscoting.A?.enabled;
       const sideB = wall.wainscoting.B?.enabled;
       if (sideA || sideB) {
         const side: WallSide = (sideA && sideB)
           ? useUIStore.getState().activeWallSide
           : sideA ? "A" : "B";
         setWainscotEditWallId(wall.id);
         setWainscotEditSide(side);
         return;
       }
     }
   }
   ```
   Import `closestPointOnWall` and `distance` from `@/lib/geometry` (add to existing imports).

3. Compute popover position (add after the existing `overlayStyle` block, ~line 363):
   ```typescript
   let wainscotOverlayStyle: React.CSSProperties | null = null;
   if (wainscotEditWallId) {
     const wall = getActiveRoomDoc()?.walls[wainscotEditWallId];
     const wrapper = wrapperRef.current;
     if (wall && wrapper) {
       const rect = wrapper.getBoundingClientRect();
       const { userZoom, panOffset } = useUIStore.getState();
       const { scale, origin } = getViewTransform(
         room.width, room.length, rect.width, rect.height, userZoom, panOffset

       };
     }
   }
   ```
