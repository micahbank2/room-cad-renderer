# v1.7.5 Design System & UI Polish — Milestone Archive

**Shipped:** 2026-04-22
**Goal:** Raise the visual + interaction quality bar to match Pascal Editor-class chrome. Unify typography, collapse panel clutter, add lightweight affordances (floating selection toolbar, gesture hints, rotation presets, inline-editable titles) and normalize library card + spacing/iconography across the app. Polish only — no new capabilities.

**Source:** Pascal Editor competitive audit (`.planning/competitive/pascal-audit.md`) — 8 concrete polish items scoped in GH milestone [v1.7.5](https://github.com/micahbank2/room-cad-renderer/milestone/8) as issues #83–#90.

## Key Accomplishments

1. **Canonical design tokens** — typography 5-tier ramp (`--text-display`/`--text-base`/`--text-sm`), spacing 7-step scale, 3-step radius in Tailwind v4 `@theme` block. `--radius-lg` bumped 6→8px. `lucide-react` installed for new UI chrome; Material Symbols pinned to 8-file CAD-domain allowlist (D-33).
2. **Mixed-case typography hierarchy** — section headers, panel headers, button labels, tab labels all converted from blanket UPPERCASE to mixed case. UPPERCASE preserved for dynamic CAD identifiers, status strings, unit value labels, tool mode labels, brand (D-03, D-04, D-05).
3. **Zero-arbitrary spacing sweep** — Toolbar, Sidebar, PropertiesPanel, RoomSettings now have zero `p-[Npx]` / `m-[Npx]` / `rounded-[Npx]` / `gap-[Npx]` arbitrary values. All spacing routes through canonical tokens.
4. **Reusable primitives** — `useReducedMotion` hook (guards all new animations per D-39), `CollapsibleSection` with localStorage persistence (10 PropertiesPanel sections wrapped), `LibraryCard` + `CategoryTabs` (ProductLibrary + CustomElementsPanel migrated), `InlineEditableText` (mirrors Phase 31 `LabelOverrideInput` pattern).
5. **FloatingSelectionToolbar** — appears above selection bbox with Duplicate + Delete actions; `uiStore.isDragging` bridge hides toolbar during drag. Single-undo contract preserved.
6. **GestureChip** — dismissible 2D + 3D variants showing pan/rotate/zoom hints; localStorage dismissal persists across reloads; hidden during active drag.
7. **Rotation preset chips** — `[-90°, -45°, 0°, +45°, +90°]` chips beside the rotation numeric input in PropertiesPanel; each click pushes exactly one undo entry (store-level behavior test enforced).
8. **Inline-editable titles** — document title (Toolbar) and room tab labels are now click-to-edit with Enter/blur commit + Escape cancel (Phase 31 `LabelOverrideInput` pattern). ProjectManager demoted to read-only display.
