  <what-built>All six POLISH features for the v1.2 Polish Pass phase</what-built>
  <how-to-verify>
Run `npm run dev` and open http://localhost:5173 in the browser. Test each feature:

**POLISH-01 — Custom element edit handles:**
1. In the sidebar CUSTOM_ELEMENTS section, create a custom element (e.g., "RUG", box, 4x6 ft)
2. Click PLACE to drop it on the canvas
3. Click the placed element — rotation handle (purple line + circle) and 4 corner handles should appear
4. Drag the element to move it
5. Drag the rotation handle to rotate it (hold Shift for 15-degree snap)
6. Drag a corner handle to resize it
7. Press Delete to remove it

**POLISH-02 — Wainscot inline edit:**
1. Open WAINSCOT_LIBRARY in the sidebar
2. Create a new wainscot style if none exist
3. Double-click the style in the library list — name, height, and color editors should appear inline
4. Change the name, height, or color — changes should persist
5. Click DONE or press Enter/Escape to exit edit mode

**POLISH-03 — Copy wall side:**
1. Draw a wall and select it
2. In the WALL_SURFACE panel, apply wallpaper/paint/wainscoting/art to SIDE_A
3. Click the "COPY_TO_SIDE_B" button
4. Switch to SIDE_B — all treatments from SIDE_A should now appear on SIDE_B

**POLISH-04 — Frame color override:**
1. Add a framed art piece from the library to a wall
2. In the wall art list, find the color picker next to the art item
3. Change the color — the frame should render in the new color in 3D view
4. Verify the original library entry is unchanged

**POLISH-05 — Multi-select + bulk paint:**
1. Draw 3+ walls
2. Cmd+click (Mac) or Ctrl+click (Win) to select multiple walls
3. Properties panel should show "N ITEMS_SELECTED (M WALLS)" with BULK_ACTIONS header
4. Pick a color in PAINT_ALL_WALLS — all selected walls should change color
5. Cmd+click a selected wall to deselect it from the group

**POLISH-06 — Sidebar scroll/collapse:**
1. Scroll the sidebar — all content should be accessible, no cutoff
2. Click section headers (ROOM_CONFIG, SYSTEM_STATS, etc.) — sections should collapse/expand
3. Click the collapse arrow in the sidebar header — entire sidebar should disappear
4. Canvas should expand to fill the freed space
5. Click the hamburger button to re-open the sidebar
