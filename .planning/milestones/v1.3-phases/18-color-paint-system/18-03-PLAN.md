    **Create src/components/SwatchPicker.tsx — shared swatch browsing component:**

    This component encapsulates all the paint browsing UI that is identical between walls and ceilings: recently-used row, hue filter chips, search, F&B catalog grid, custom colors section with add/delete.

    **Props interface:**
    ```typescript
    interface Props {
      activePaintId?: string;          // currently applied paintId (for ring highlight)
      onSelectPaint: (paintId: string) => void;  // callback when user clicks a swatch
    }
    ```

    **Component structure (top to bottom, per UI-SPEC Component Inventory):**

    1. **RECENTLY_USED row (per D-04, UI-SPEC):**
       - Label: `<span className="font-mono text-[9px] tracking-widest uppercase text-text-dim">RECENTLY_USED</span>`
       - Read `recentPaints` from cadStore: `const recentPaints: string[] = useCADStore((s) => (s as any).recentPaints ?? []);`
       - Read `customColors` from paintStore: `const customColors = usePaintStore((s) => s.customColors);`
       - For each paintId, resolve hex via `resolvePaintHex(id, customColors)` and render an 18x18px swatch
       - Swatch: `<button className="w-[18px] h-[18px] rounded-sm" style={{ backgroundColor: hex }} onClick={() => onSelectPaint(id)} />`
       - If empty: `<span className="font-mono text-[9px] text-text-ghost">NO_RECENT_COLORS</span>`
       - Container: `<div className="flex gap-1 flex-wrap">`

    2. **HUE_FILTER chips (per D-03, UI-SPEC):**
       - Label: `<span className="font-mono text-[9px] tracking-widest uppercase text-text-dim">HUE_FILTER</span>`
       - For each family in HUE_FAMILIES, render a 14x14px chip with a representative color:
         ```
         WHITES: "#f4f0e8", NEUTRALS: "#b5a99a", BLUES: "#2d4a6e",
         GREENS: "#6b7c5e", PINKS: "#c7918a", YELLOWS: "#d4a84b", BLACKS: "#2a2a2a"
         ```
       - Active chip: `border border-accent text-accent-light bg-accent/10`
       - Inactive chip: `border border-outline-variant/30 text-text-ghost`
       - State: `const [activeHue, setActiveHue] = useState<string | null>(null);`
       - Click toggles: if same hue clicked again, clears filter (set null)
       - Container: `<div className="flex gap-1 items-center flex-wrap">`

    3. **SEARCH_COLORS input (per UI-SPEC):**
       - `<input type="text" placeholder="SEARCH_BY_NAME" className="w-full bg-obsidian-high border border-outline-variant/20 rounded-sm px-2 py-1 font-mono text-[9px] text-text-primary placeholder:text-text-ghost focus:border-accent/50 outline-none" />`
       - State: `const [search, setSearch] = useState("");`

    4. **F&B_CATALOG swatch grid (per D-02, UI-SPEC — PRIMARY FOCAL POINT):**
       - Label: `<span className="font-mono text-[9px] tracking-widest uppercase text-text-dim">F&B_CATALOG ({filteredCount})</span>`
       - Filter logic: `FB_COLORS.filter(c => (!activeHue || c.hueFamily === activeHue) && (!search || c.name.toLowerCase().includes(search.toLowerCase())))`
       - Grid container: `<div className="grid grid-cols-8 gap-1 max-h-40 overflow-y-auto">`
       - Each swatch: 20x20px button with `style={{ backgroundColor: c.hex }}`
       - Applied swatch (matches activePaintId): `ring-2 ring-accent ring-offset-1 ring-offset-obsidian-low`
       - Hover: `ring-1 ring-white/30`
       - Tooltip on hover (300ms delay): show F&B name in `font-mono text-[8px] text-text-primary bg-obsidian-highest border border-outline-variant/30 rounded-sm px-1 py-1`. Implement with `onMouseEnter` + `setTimeout` + `onMouseLeave` clear.
       - Click: calls `onSelectPaint(c.id)`
       - Empty result: `<div className="text-center py-2 font-mono text-[9px] text-text-ghost">NO_COLORS_FOUND</div>`

    5. **MY_COLORS section (per D-10, D-11, D-12, UI-SPEC):**
       - Label: `<span className="font-mono text-[9px] tracking-widest uppercase text-text-dim">MY_COLORS</span>`
       - Render custom swatches same as F&B but 20x20px in a flex row
       - Right-click on custom swatch: show micro context menu with `DELETE` option
         - `onContextMenu={(e) => { e.preventDefault(); setDeleteMenuId(c.id); setMenuPos({x: e.clientX, y: e.clientY}); }}`
         - Menu: `<div className="fixed z-50 bg-obsidian-highest border border-outline-variant/30 rounded-sm px-2 py-1"><button className="font-mono text-[9px] text-error" onClick={() => { removeCustomPaint(c.id); setDeleteMenuId(null); }}>DELETE</button></div>`
         - Close menu on click outside (useEffect with document click listener)
         - `removeCustomPaint` comes from cadStore: `const removeCustomPaint = useCADStore((s) => s.removeCustomPaint);`
       - If no custom colors: `<span className="font-mono text-[9px] text-text-ghost">NO_CUSTOM_COLORS</span>`
       - `+ ADD_COLOR` button: `<button className="font-mono text-[9px] text-text-dim hover:text-accent-light">+ ADD_COLOR</button>`
       - State: `const [showAddForm, setShowAddForm] = useState(false);`

    6. **Add custom color inline form (per D-11, D-09):**
       - Only visible when `showAddForm === true`
       - Layout: column with name input, react-colorful `HexColorPicker` (w-40), SAVE_COLOR button
       - Name input: `<input placeholder="COLOR_NAME" className="w-full bg-obsidian-high border border-outline-variant/20 rounded-sm px-2 py-1 font-mono text-[9px] text-text-primary placeholder:text-text-ghost" />`
       - Import `HexColorPicker` from `react-colorful`
       - Picker: `<HexColorPicker color={newHex} onChange={setNewHex} style={{ width: "160px", height: "120px" }} />`
       - State: `const [newName, setNewName] = useState(""); const [newHex, setNewHex] = useState("#7c5bf0");`
       - SAVE_COLOR button disabled when newName is empty:
         - Disabled: `className="font-mono text-[9px] text-text-ghost cursor-not-allowed"`
         - Enabled: `className="font-mono text-[9px] text-accent-light hover:text-accent cursor-pointer"`
       - On save: call cadStore action directly: `addCustomPaint({ name: newName, hex: newHex, hueFamily: undefined }); setShowAddForm(false); setNewName(""); setNewHex("#7c5bf0");`
         - `const addCustomPaint = useCADStore((s) => s.addCustomPaint);`
       - Enter key in name field triggers save if name non-empty


       - `handleToggleLimeWash(checked: boolean)`: calls `setWallpaper(wallId, side, { ...currentWallpaper, kind: "paint", limeWash: checked })`

    **Handler functions:**
