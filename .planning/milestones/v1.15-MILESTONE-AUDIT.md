# v1.15 Architectural Toolbar Expansion — Milestone Audit

**Goal of the milestone:** Make the *room* as rich as the furniture is. Jessica can now visualize the inside of a room from any angle (cutaway), add stairs, place archways/passthroughs/niches between rooms, and measure + label spaces with auto room-area in the Properties panel.

**Phase status:**
- Phases 59, 60, 61 → merged to `main`.
- Phase 62 → PR [#145](https://github.com/micahbank2/room-cad-renderer/pull/145) OPEN, all gates green, ready to merge.
