## v1.2 Requirements (Shipped)

### Ceilings

- [x] **CEIL-01**: Jessica can draw a ceiling as an independent polygon surface on the 2D canvas — shipped via ceilingTool + addCeiling store action
- [x] **CEIL-02**: Each ceiling has an editable height (defaults to room's wallHeight) — updateCeiling action
- [x] **CEIL-03**: Ceiling renders in 3D as a flat surface overhead — CeilingMesh component
- [x] **CEIL-04**: Ceiling has an optional material (hex color) — material field on Ceiling type (preset-id path documented but deferred, hex works)

### Floor Materials

- [x] **FLOOR-01**: 8-preset catalog (wood-oak, wood-walnut, tile-white, tile-black, concrete, carpet, marble, stone) — FLOOR_PRESETS in floorMaterials.ts
- [x] **FLOOR-02**: Custom floor texture upload (data URL) — FloorMaterialPicker file input
- [x] **FLOOR-03**: Editable scale + rotation for pattern tiling — scaleFt + rotationDeg on FloorMaterial

### Wall Surfaces

- [x] **SURFACE-01**: Per-wall wallpaper (tiled pattern image or solid color) — Wallpaper type + setWallpaper action (made per-side in Phase 17)
- [x] **SURFACE-02**: Wall art items placed on any wall at specific x-position + height — WallArt[] + addWallArt action
- [x] **SURFACE-03**: Configurable width + height, rendered in 3D attached to wall face — WallMesh art rendering path
- [x] **SURFACE-04**: Moveable, resizeable, removable post-placement — updateWallArt + removeWallArt

### Architectural Trim

- [x] **TRIM-01**: Per-wall wainscoting toggle with editable height (default 36") — WainscotConfig + toggleWainscoting (made per-side in Phase 17)
- [x] **TRIM-02**: Per-wall crown molding toggle — CrownConfig + toggleCrownMolding
- [x] **TRIM-03**: Wainscoting + crown render as 3D architectural bands — renderSideDecor in WallMesh

### Custom Element Builder

- [x] **CUSTOM-01**: Builder with name, shape (box/plane), W×D×H, color — CustomElementsPanel + addCustomElement
- [x] **CUSTOM-02**: 2D canvas placement without image required — renderCustomElements in fabricSync
- [x] **CUSTOM-03**: Edit handles (rotate/resize/move) — deferred to v1.3 (uses product select hit-testing currently)
- [x] **CUSTOM-04**: 3D render at configured dimensions + material — CustomElementMesh
- [x] **CUSTOM-05**: Per-project catalog persistence + cascade on delete — saveProject includes customElements (PR #15 fix)

### Framed Art Library

- [x] **ART-01**: ART_LIBRARY sidebar section — FramedArtLibrary component
- [x] **ART-02**: 7 frame presets — FRAME_PRESETS (none, thin-black, thick-gold, natural-wood, museum-white, floating, ornate)
- [x] **ART-03**: Library persists across projects — framedArtStore (IndexedDB key: room-cad-framed-art)
- [x] **ART-04**: Placed on any wall via + LIB picker in WallSurfacePanel
- [x] **ART-05**: 3D frame geometry (4 protruding strips + inset art plane) — WallMesh framed art path
- [x] **ART-06**: Width + height editable post-placement — updateWallArt action

### Per-Side Wall Treatments (Added mid-milestone)

- [x] **SIDE-01**: Every wall treatment (wallpaper, wainscoting, crown, wall art) independently configurable per face — WallSegment per-side wrappers
- [x] **SIDE-02**: SIDE_A / SIDE_B toggle UI in WallSurfacePanel — uiStore.activeWallSide
- [x] **SIDE-03**: 3D renders both sides independently — renderSideDecor called twice (side B in 180° Y-rotation group)

### Wainscoting Style Library

- [x] **WAIN-01**: 7 wainscoting styles (recessed-panel, raised-panel, beadboard, board-and-batten, shiplap, flat-panel, english-grid) — wainscotStyles.tsx renderers
- [x] **WAIN-02**: Configurable panel width, stile width, chair rail height, depth — per-style knob map in STYLE_META
