## Phases

### Phase 49: Wall Texture First-Apply Fix (BUG-02)
**Goal:** Uploaded wall textures render in 3D on first apply without requiring a view-mode toggle.
**Requirements:** BUG-02
**Plans:** 1/1 complete
**Root cause:** `<primitive attach="map">` did not set `material.needsUpdate=true` when texture transitioned from null → Texture.
**Fix:** Replace with direct `map={userTex}` prop on `<meshStandardMaterial>` — eliminates the requirement at root.

### Phase 50: Wallpaper/WallArt View-Toggle Persistence (BUG-03)
**Goal:** Custom-uploaded wallpaper and wallArt survive 2D↔3D view-mode switches.
**Requirements:** BUG-03
**Plans:** 1/1 complete
**Scope discovery:** Research found Phase 49's fix incidentally fixed user-uploaded wallpaper (shared `useUserTexture` path). Only wallArt remained broken.
**Fix:** Apply Phase 49's direct-prop pattern to two wallArt sites (unframed + framed-inner).

### Phase 51: Legacy FloorMaterial Snapshot Migration (DEBT-05)
**Goal:** Saved snapshots no longer embed raw data-URL strings for custom floor textures.
**Requirements:** DEBT-05
**Plans:** 1/1 complete (heaviest of v1.12)
**Architectural change:** `loadSnapshot` is now `async (raw: unknown) => Promise<void>`. 23 caller sites updated (7 production + 3 vitest + 12 e2e + 1 shared helper).
**Migration:** Idempotent v2→v3 schema bump. Async pre-pass before Immer `produce()`. SHA-256 dedup via existing `saveUserTextureWithDedup`. Graceful on malformed data URLs (preserved + console.warn).

### Phase 52: Keyboard Shortcuts Overlay (HOTKEY-01)
**Goal:** Pressing `?` opens a keyboard shortcuts cheat sheet overlay listing all hotkeys grouped by category.
**Requirements:** HOTKEY-01
**Plans:** 1/1 complete
**Discovery:** HelpModal infrastructure was 80% built. Phase 52 finished the 20%.
**Built:** `src/lib/shortcuts.ts` single-source-of-truth registry consumed by both App.tsx keyboard handler AND helpContent.tsx display. 26 entries (was 19 — added 7 missing: Ceiling C, Reset 0, Camera Presets 1-4, Copy Cmd+C, Paste Cmd+V).
**Bug fix:** `?` now calls `openHelp("shortcuts")` explicitly so modal lands on the right tab.

## Notable architectural changes

- **Async `loadSnapshot`** — Pattern A architecture (async pre-pass before Immer `produce()`). All future migration work inherits this.
- **Snapshot version 2 → 3** — chained migrations preserve v1→v2→v3 path.
- **Single-source-of-truth keyboard registry** — coverage-gate test prevents future drift.
- **Direct `map={tex}` prop pattern** standardized for R3F texture rendering.
