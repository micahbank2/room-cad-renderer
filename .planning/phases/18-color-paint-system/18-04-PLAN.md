<task type="checkpoint:human-verify">
  <name>Task 1: Visual verification of complete paint system</name>
  <files></files>
  <action>
    Start the dev server (`npm run dev`) and present the following verification checklist to the user.

    **PAINT-01 — Wall paint application:**
    1. Draw a room with at least 2 walls
    2. Click a wall to select it
    3. In the sidebar, find the PAINT section (below WALLPAPER)
    4. Click any F&B swatch — wall should show that color in 2D immediately
    5. Switch to 3D view — same wall shows the paint color
    6. Switch to SIDE_B using the side toggle — apply a different color
    7. Both sides should show their respective colors in 3D

    **PAINT-02 — Ceiling paint:**
    8. Draw a ceiling (or select an existing one)
    9. In the properties panel, find CEILING_PAINT section
    10. Click a swatch — ceiling shows color in 3D

    **PAINT-03 — F&B catalog browsing:**
    11. Back on a wall, verify the swatch grid shows many colors (132 total)
    12. Click a hue filter chip (e.g., BLUES) — grid should narrow to blue-family colors
    13. Click the same chip again — filter clears, all 132 show
    14. Type a color name in SEARCH_BY_NAME (e.g., "hague") — grid filters to matching names
    15. Hover over a swatch for ~300ms — tooltip should show the F&B color name

    **PAINT-04 — Custom colors:**
    16. Scroll to MY_COLORS section
    17. Click + ADD_COLOR — inline form should expand with name input + color wheel
    18. Type a name, pick a color, click SAVE_COLOR — swatch appears in MY_COLORS
    19. Click the custom swatch — applies to wall
    20. Right-click the custom swatch — DELETE option appears
    21. Click DELETE — swatch is removed

    **PAINT-05 — Lime wash:**
    22. Apply any paint color to a wall
    23. Check the LIME_WASH_FINISH checkbox
    24. In 2D: wall should show a subtle stippled/dotted overlay
    25. In 3D: wall should look noticeably more matte/chalky than without lime wash

    **PAINT-06 — Recently used:**
    26. After applying several different colors, check the RECENTLY_USED row at the top
    27. Should show the last applied colors (most recent at left, up to 8)
    28. Click a recently-used swatch — applies that color

    **PAINT-07 — Apply to all walls:**
    29. Apply any paint color to one wall
    30. Click APPLY_TO_ALL_WALLS button
    31. All walls in the room should now show that color on the active side
    32. Check 3D view — all walls show the paint color

    Wait for user to type "approved" or describe issues.
  </action>
  <verify>User types "approved" or provides issue list</verify>
  <done>All 7 PAINT requirements confirmed working by user visual inspection in both 2D and 3D views.</done>
